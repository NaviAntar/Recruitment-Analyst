# Centralized Recruitment Monitoring & Workforce Reporting System

A centralized HR recruitment and workforce reporting platform developed to improve recruitment visibility, candidate pipeline tracking, onboarding monitoring, and reporting efficiency within PT Darma Henwa Tbk.

This project integrates recruitment operations, PRF requisition management, candidate progress tracking, and executive dashboard reporting into a unified workflow using Google Sheets, Google App Script, Looker Studio, and Streamlit. The system was designed to streamline recruitment monitoring processes, improve data accuracy, and provide real-time workforce insights for management decision-making.

# Core Features

## Recruitment Pipeline Tracking

The system tracks candidate progress across multiple recruitment stages including:

- Screening CV
- HR Interview
- User Interview
- Psychotest
- Offering
- Medical Check-Up (MCU)
- MCU Review
- Follow-Up MCU
- Onboarding

Each stage includes tracking for:

- Start date
- Completion date
- SLA monitoring
- Result status
- Delay reasons
- Candidate progress status

The recruitment progress is visualized through interactive progress tracking and completion monitoring within Streamlit dashboards.

---

## PRF Requisition Management

The system includes a centralized PRF module for monitoring manpower requests and recruitment assignments.

Key capabilities:

- PRF creation and editing
- Position metadata integration
- PRF status tracking
- Duplicate PRF validation
- Workforce request monitoring
- Recruitment assignment linking

The PRF workflow is connected directly to candidate recruitment tracking to improve manpower visibility and recruitment coordination.

---

## Dynamic Candidate Management

The platform supports:

- Multi-candidate input
- Candidate profile management
- Recruitment stage updates
- Candidate search functionality
- Recruitment status monitoring
- Bulk recruitment process handling

Users can dynamically add, modify, and monitor candidate records through integrated recruitment forms.

---

## Executive Recruitment Dashboard

Interactive recruitment dashboards were developed using Looker Studio and embedded into the Streamlit application to provide:

- Recruitment funnel analysis
- Hiring progress visibility
- Onboarding monitoring
- Recruitment SLA tracking
- Workforce reporting
- Recruitment performance insights

The dashboard supports management-level monitoring and centralized recruitment reporting.

# System Architecture

## Data Flow

### Google Sheets
Functions as the centralized database layer for:

- Recruitment progress logs
- PRF requisitions
- Backend position master data

### Google App Script
Handles:

- Data processing
- Validation
- Search engine functions
- Record creation and updates
- Workflow automation

### Streamlit
Functions as the frontend portal for:

- Recruitment monitoring
- Candidate tracking visualization
- Dashboard integration
- Recruitment progress analytics

### Looker Studio
Provides:

- Executive-level reporting
- Interactive recruitment dashboards
- Workforce visualization
